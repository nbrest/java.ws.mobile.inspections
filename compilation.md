# Compilation:

* Compile using `mvn clean install [compilation option]` in the root to build all the modules.

| Option | Usage | Description | 
| ------------------ | ----- | ----------- |
| P | -P:prod | Uses mysql db. **Default**  |
|   | -P:qa | Uses oracle db |
|   | -P:dev | Uses hsql in memory db |
|   | -P:docker | For docker images |
|   | -P:ci | For jenkins |
