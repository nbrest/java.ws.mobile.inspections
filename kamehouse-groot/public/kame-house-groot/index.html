What is GRoot
GRoot is a webapp that runs on an plain http server such as apache httpd and its few API's are built with php

The APIs are mainly to get the current httpd session status and to execute shell scripts in the server from the KameHouse Shell module

Some things you can do with GRoot

  • Start/Stop/Deploy/Undeploy all kamehouse modules
  • Git pull to keep all repos updated
  • Update media database on the Media Server
  • Trigger all those commands in all servers running kamehouse at once
  • Restart tomcat running kamehouse
  • Restart the server running kamehouse
